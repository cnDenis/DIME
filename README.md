# 迪弥五笔输入法(DIME)

迪弥五笔输入法是基于Windows TSF IME的五笔98输入法.

缘起是因为我用了十几年的小鸭五笔不更新了. 现在的Windows系统里有些地方, 比如说命令行窗口, 用小鸭是输入不了的. 必须要使用基于TSF框架的输入法才可以输入. 然而现有输入法对于五笔98的支持一般, 也很难调教出我习惯的用法. 所以在AI的帮助下, 我基于微软的TSF示例代码, 做了这个输入法.

主要的功能都是仿小鸭五笔的, 如下:

- z键开启临时拼音输入, 临时拼音的会显示字的五笔编码
- ;键(分号)或者大写字母开启临时英文输入, 回车上屏
- `Ctrl+M` 切换常用字表(GB2312)和整个码表检索, 在状态栏上显示为 常/全
- `Ctrl+.` 切换中英文标点
- `Shift+空格` 切换全半角

因为是满足我个人使用的需要, 我自己用不上的功能一概没有. 有需要的可以把源代码丢给AI让AI帮忙做.

自带五笔码表是海峰98码表. 十几年前不知道从哪里下载回来的了, 版权归于原作者. 码表可以自己换, 替换安装目录下dict文件夹里的txt文件, 然后运行一下build_bindict.exe, 就可以生成对应码表文件.
